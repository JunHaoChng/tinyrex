# Following tutorial:
https://linuxhint.com/lamp_server_docker/

## Running instructions for LAMP
### Installation
```
cd lamp
docker-compose up
```

## Useful Commands
To see port mapping after <code>docker-compose up</code>
```
docker-compose ps
```

## Visit the PHP admin
```
http://localhost:5000/
#or
http://YOURIPADDRESS:5000/
```
